# FabricJS 7 + React + TypeScript + Vite

[FabricJS](https://fabricjs.com/) is a versatile library for drawing on an HTML
5 canvas. Unfortunately, the documentation of the latest FabricJS version, 7,
is lacking basic examples.

For a professional project I needed to properly understand how to do basic (and
slightly more advanced) things in a technology stack consisting of FabricJS 7 +
React + TypeScript + Vite. Therefore, I've created a small demo web app that
you can find
[here](https://robvanderleek.github.io/fabricjs7-react-typescript-vite/) (see
screenshot below).
